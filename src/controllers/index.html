Zoom to YouTube Uploader

Application for the Hexlet team. Downloads recordings from one Zoom account and uploads them to one YouTube channel

This application cannot be used by anyone except Hexlet employees. The application does not have an external interface and works only in server-to-server mode.

Privacy policy: https://downloader.hexlet.io/privacy

The application is supported by Hexlet employees. For any questions, you can write to support at support@hexlet.io

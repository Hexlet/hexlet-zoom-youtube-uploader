Privacy Policy of Zoom to YouTube Uploader

This application is for a limited number of users. It can only be used by members of the Hexlet team. The application does not have an external interface and works only in server-to-server mode.

The application requires the following accesses:
* Manage your YouTube videos
* View your YouTube account
* Manage your YouTube account

These accesses are used to:
* Add new videos to the channel
* Create new playlists on the channel and add videos to them

The application has no more functionality.

The application stores in the database:
* OAuth token for adding videos
* links to saved videos

No one has access to the saved data, it is not transferred to third parties, and is not used by the application developer.

The application is supported by Hexlet employees. For any questions, you can write to support at support@hexlet.io
